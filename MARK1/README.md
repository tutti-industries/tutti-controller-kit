# MARK1

ツッチーコントローラーキット MARK1 の資料一式です。

## 構成

- `common/` : 回路図・部品表・共通の取扱説明
- `STAGE1/`〜`STAGE4/` : 各STAGEごとの手順書・コード

各STAGEフォルダをGitHub上でダウンロード（フォルダの「...」メニューからDownload）すれば、そのSTAGEに必要な手順書PDFとコードが一括で手に入ります。

## STAGE一覧

| STAGE | 内容 | firmware |
|---|---|---|
| [STAGE1](STAGE1) | はんだ付けの練習 | なし |
| [STAGE2](STAGE2) | オームの法則の理解 | なし |
| [STAGE3](STAGE3) | 半導体（トランジスタ）の理解 | あり |
| [STAGE4](STAGE4) | センサー（プッシュスイッチ・ジョイスティック）の理解 | あり |
